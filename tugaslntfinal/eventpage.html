Daftar Acara

  • Acara Pembukaan

    • Untuk acara pembukaan, para pembicara yang terhormat akan memperkenalkan diri dan CEO PT. Meksiko akan mengucapkan beberapa kata sebelum memulai event selanjutnya.

  • Event Interview

    • Untuk event interview, pembicara akan mengadakan sesi interview dengan interviewer. Setelah selesai, pembicara akan menerima Q&A dari penonton.

  • Event Meet & Greet

    • Untuk event meet and greet, penonton akan berkesempatan untuk bertemu secara langsung dan foto dengan pembicara jika mereka menginginkannya.

  • Event Showcase

    • Untuk event showcase, pembicara akan menunjukkan kreasi-kreasi terbarunya dan membiarkan para penonton mencicipinya.

  • Acara Penutup

    • Untuk acara penutup, CEO PT. Meksiko akan memberikan salam dan foto dengan pembicara dan panitia akan mengadakan door prize.
